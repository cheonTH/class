- overriding은 부모클래스를 상속 받을 때 메서드를 상속받아 재정의하는 것 
- overloading은 같은 클래스에서 같은 이름의 메서드를 사용하기 위해 매개변수의 개수를 다르게 하는 것

2.
클래스, 메서드, 변수에 대해 변경 불가능을 지정하는데 사용
클래스: 다른 클래스에서 상속(extends) 불가능
메서드: 오버라이딩 불가능
변수: 지정된 값에 대한 변경 불가능

3.
자바에서 예외처리란 프로그램 실행 시 발생할 수 있는 오류의 예외 발생에 대비한 코드를 작성하는것
try부분: 오류가 발생할 수 있는 명령어를 작성
catch부분: 지정한 오류가 발생했을 때 처리할 명령어를 작성
finally부분: 예외처리와 상관없이 무조건 실행되어야하는 명령어 작성
