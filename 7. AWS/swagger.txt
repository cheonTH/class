Swagger
Restful API의 명세를 작성, 시각화, 테스트할 수 있는 프레임워크
현재는 Swagger 사양이 발전하여 OpenAPI Specification으로 통합되었으며, Swagger는 OpenAPI구현체중
하나임

사전적인 의미로서의 OpenAPI는 개방된 API임 누구나 사용할 수 있도록 엔드포인트가 열린 API

OpenAPI 또는 OAS라고 부르는데, Restful API를 정의된 규칙에 맞게 Api spec을 json이나 yaml로 표현하는
방식, 직접 소스코드나 문서를 보지 않고 서비스를 이해할 수 있다는 장점이 있음

Swagger가 독립적으로 존재했으나
