<<UI/UX 콘셉트 기획>>

UI:사용자가 직접 보는 화면 디자인
UX:사용자가 서비스를 이용하면서 경험하는 모든 과정
중요한 이유
사용자의 문제를 해결 할 수 있도록 서비스를 설계한는 과정

콘셉트 기획을 하는 이유
-사용자의 불편함을 해결할 수 있도록 서비스를 설계하는 과정

사용자 시나리오
-사용자가 서비스를 이용하는 과정을 시나리오 형태로 정리하는 단계

1. 사용자 목표 정의
- 사용자가 원하는 것은 무엇인지 정리
2. 사용자의 행동 플로우 작성
- 예를 들어 "사용자가 로그인 후 상품을 검색하고 결제하는 과정" 정리
3. 각 단계에서의 사용 경험 개선
- 불편한 요소가 없는지 확인하고 UI 개선점을 정리

사용자 스토리(User Story) 작성
- 사용자가 원하는 기능을 한 문장으로 정리하는 방법
- 사용자 역할: 온라인 쇼핑몰 사용자
- 필요한 기능: 상품을 장바구니에 담을 수 있음
- 이유: 한번에 여러 개의 상품을 결제하고 싶어서

스토리 보드(Story board) 작성
-사용자 시나리오를 시각적으로 표현
-UI설계를 구체적으로 할 수 있고, 개발자와 디자이너가 쉽게 이해할 수 있음
